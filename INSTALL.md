# Quick Installation Guide

## Step 1: Convert Icons to PNG

Before loading the extension, you need to convert the SVG icons to PNG format:

1. Open `icon-converter.html` in your browser (just double-click it)
2. Click the "Download PNG" button for each icon size (128, 48, 16)
3. Save all three PNG files into the `icons/` folder

## Step 2: Load Extension in Chrome

1. Open Google Chrome
2. Navigate to `chrome://extensions/`
3. Enable "Developer mode" (toggle in top-right corner)
4. Click "Load unpacked"
5. Select this folder: `Kindle Highlighter`

## Step 3: Test the Extension

1. Go to [Kindle Cloud Reader](https://read.amazon.in/) or [read.amazon.com](https://read.amazon.com/)
2. Open any book
3. Select some text
4. Try the default shortcuts:
   - **Yellow**: `Ctrl+Shift+Y` (Mac: `Cmd+Shift+Y`)
   - **Blue**: `Ctrl+Shift+B` (Mac: `Cmd+Shift+B`)
   - **Pink**: `Ctrl+Shift+P` (Mac: `Cmd+Shift+P`)
   - **Orange**: `Ctrl+Shift+O` (Mac: `Cmd+Shift+O`)

## Step 4: Customize (Optional)

1. Click the extension icon in your Chrome toolbar
2. Click "Customize Shortcuts"
3. Set your preferred keyboard shortcuts
4. Click "Save Settings"

## Troubleshooting

### Icons not showing in Chrome?
- Make sure you've converted all three SVG files to PNG
- PNG files should be named: `icon16.png`, `icon48.png`, `icon128.png`
- They should be in the `icons/` folder

### Shortcuts not working?
- Ensure you're on a Kindle Cloud Reader page
- Make sure text is selected
- Check for shortcut conflicts at `chrome://extensions/shortcuts`
